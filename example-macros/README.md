# Example AutoHotkey Macros

This directory contains example AutoHotkey (.ahk) and AutoHotkey Macro (.amc) files that were included with the project.

## Directory Structure

- **Root Files**: Individual macro files (.ahk, .amc, .txt)
- **archives/**: Compressed archives (.zip) containing additional macros and scripts

## Usage

These files serve as examples and reference material for:
- Testing the macro search functionality
- Understanding AutoHotkey script structure
- Providing sample macros for users

## Note

These are example files and may require modification to work with your specific setup or game version.
